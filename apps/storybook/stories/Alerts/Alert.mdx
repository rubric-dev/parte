# Alert

**Alert** 는 사용자에게 피드백을 주기 위한 컴포넌트입니다.

**Toaster** 는 내부적으로 **Alert** 를 컴포넌트를 사용하고 있습니다.

## Props

**Alert** 컴포넌트의 공통 속성은 `type`, `status` 2가지가 존재합니다.

`type` 이 inline 일때는 `message` 에 원하는 텍스트를 넘겨주면 됩니다.

`type` 이 alert, toast 일 경우에는 `title`, 그리고 설명은 `children` 형식으로 넘겨주면 가능합니다. toast type 의 경우 닫기 버튼을 위한 onClose 콜백 함수를 넘겨줄 수 있습니다
